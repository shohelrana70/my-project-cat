Question 2: What is the difference between innerHTML, innerText, and textContent?

Answer : Use innerHTML for HTML content, innerText for visible text, and textContent for all text regardless of visibility.

Question 3: What is event delegation in the DOM?

Answer : Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners on individual child elements

Question 4: What is event bubbling in the DOM?

Answer : Event bubbling is a mechanism in the DOM where an event propagates upward from the target element to its ancestors in the hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer : document.createElement(tagName). To add an element to the DOM, use methods like parentElement.appendChild(newElement) or parentElement.insertBefore(newElement, referenceElement). To remove an element, use element.parentNode.removeChild(element) or element.remove().
